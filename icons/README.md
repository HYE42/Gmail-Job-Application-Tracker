# Extension Icons

## How to Generate Icons

### Option 1: Using the HTML Generator (Easiest)
1. Open `create-icons.html` in your web browser
2. Click each canvas to download the corresponding PNG file
3. Save them in this directory as:
   - icon16.png
   - icon48.png
   - icon128.png

### Option 2: Create Your Own Icons
Create three PNG files with the following specifications:
- **icon16.png**: 16x16 pixels (toolbar icon)
- **icon48.png**: 48x48 pixels (extension management page)
- **icon128.png**: 128x128 pixels (Chrome Web Store)

### Design Guidelines
- Use a simple, recognizable design
- Blue color scheme (#4285f4) to match the extension theme
- Envelope or briefcase icon to represent job applications
- Ensure good visibility at 16x16 size

### Placeholder Icons
If you don't have custom icons yet, you can use simple colored squares as placeholders:
- The extension will still function without custom icons
- The browser will display a default icon if none are provided

## Current Status
Icons need to be generated using the HTML file provided or created manually.
